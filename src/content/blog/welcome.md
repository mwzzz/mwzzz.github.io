这是「敲码小站」的新起点。站点不再只是传统博客列表，而是一个**个人学习工作台**：首页汇总最近笔记、学习方向和进行中的项目。

## 这个站能做什么

- 用 Markdown 写学习笔记
- 按标签浏览主题
- 在首页快速跳到最近内容与项目

## 怎么写新笔记

在 `src/content/blog/` 新建 Markdown 文件，填好 frontmatter，然后本地预览：

```bash
npm run dev
```

推送到 `main` 后，GitHub Actions 会自动构建并发布。

> 坚持写作，持续进步。
